# Ollama Chat UI

A ChatGPT-style chat interface for local Ollama LLMs with Stable Diffusion image generation.

## Setup (3 commands)

```bash
git clone https://github.com/TheWarrior-tech/ollama-chat-ui
cd ollama-chat-ui
docker compose up --build
```

Open **http://localhost:3000**

> Make sure Ollama is running first: `ollama serve`
